### Design and Build Your OWN OS

## See MANIFEST.md for Project Details!

# 1. Put your favorite Kernel/Filesystem files (the CORE) in the /kernel subfolder (instructions in that folder).

# 2. Put your OS Asset Pack in the /assets subfolder (instructions in that folder).

# 3. Put your featured images in the /images subfolder. icon.png is a good name for the branding.

# 4. Edit the sample GENOME.yaml and USER_CONFIG.yaml files with your own preferences.

# 5. Build Your Operating System using the Compiler (Uses C/ASM/Python/Bash Shell).

**FILESYSTEM STRUCTURE**

/{USER}//ANTHROHEART/ANTHRO/FurryOS/AnthroHeart_Studio_in_a_Box(1)

**Requirements**

1. Four User Levels: Granny, Normal, Gamer, Hacker, Paranoid
2. "Furry" is a Mode that can be enabled.

**Notes**

1. Granny Mode: Everything is Automated, even can include encryption.
